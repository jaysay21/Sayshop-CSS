	Title: JS Computer Shop

description:
	
	this system is intended for a computer shop to manage their inventory and transactions

features:

	1. The admin can log in and logout.
	2. This system has an Inventory Page, This page is for viewing available items on the shop that can be sold, The admin can insert
            new items.
	3. This system has a Transaction Page. This page can view the customers who bought items in the shop, the item
	that was bought is updated on when it was bought and its warranty,The item that was bought is subtracted on the qunatity of the item
		in the inventory.
